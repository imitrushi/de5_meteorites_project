Cleaning and analysing data using R language

Background and scope - Our aim for this project is primarily didactic; making use of what we have been learning in our course to manipulate and analyse a data set.
Programin language  - The programing language we use is the R language. In particular the "tidyverse" library.
Data used - A subset of NASA's "Meteorites Landings" data set found at this URL https://data.nasa.gov/Space-Science/Meteorite-Landings/gh4g-9sfh/data
